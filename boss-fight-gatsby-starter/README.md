# Boss Fight Gatsby Starter

Starter pronto para o repositório `Boss-fight`, com:

- Gatsby configurado para GitHub Pages
- `pathPrefix` ajustado para `/Boss-fight`
- workflow de deploy via GitHub Actions
- landing page temática de RPG
- pasta `static/game/` para você adicionar a build do jogo
- pasta `static/prints/` para seus screenshots

## Como usar

1. Extraia o conteúdo deste `.zip` na raiz do repositório.
2. Adicione os arquivos do jogo em `static/game/`.
3. Adicione os prints em `static/prints/`.
4. Rode `npm install`.
5. Rode `npm run develop` para testar.
6. Faça commit e push para a branch `main`.
